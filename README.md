# Finex Core

.-. just use for finex and applogic (ZSmartex version not OpenWare version)

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  finex-core:
    github: zsmartex/finex-core
```

### Requirements
- crystal `>= 0.35.0`.

## Usage

```
Finex.logger.debug { "Hi" } 
Finex.cache
```
